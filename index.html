Voltar

  • HTML 5

HTML é a linguagem de marcação de texto utilizada para dar conteúdo ao seu site!
O CSS é a tecnologia utilziada para deixar seu site mais interativo e estiloso, com o design que for melhor pra você!

Aqui também faço uso do SCSS e do SASS, comumente conhecidos como "CSS com super poderes" para potencializar ainda mais a criação de estilos.
O Javascript é hoje em dia a tecnologia mais utilizada para web.
Este é responsável por aprimorar a experiência interativa do usuário com a página, e adicionar todas as funcionalidades que a aplicação exige como interações e cálculos.
PHP é uma linguagem especializada para o back-end, ou seja, é utilizado para desenvolver funcionalidades que estarão "por baixo dos panos".

No back-end são tratadas questões como a segurança da sua aplicação, relacionamento com banco de dados, dentre outros.

Com o PHP também utilizo tecnologias como Doctrine ORM para melhorar o contato com o banco de dados, além do framework Laravel.
